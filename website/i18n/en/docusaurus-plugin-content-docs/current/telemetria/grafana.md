# Grafana

## Description

Grafana visualizes the metrics stored in Prometheus through pre-configured dashboards.

## Access

- **URL**: http://localhost:3030
- **Credentials**: admin/admin
- **Management IP**: 10.99.1.11

## Pre-configured Dashboards

| Dashboard | Description |
|-----------|-------------|
| SROS Dashboard | BNG MASTER and SLAVE metrics |
| Small ISP SR Linux Edge | OLT, Carrier 1, and Carrier 2 metrics |
| Nokia Syslog Overview | SR OS and SR Linux logs through Alloy + Loki |
