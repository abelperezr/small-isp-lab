# Prometheus

## Descripción

Prometheus almacena las métricas recolectadas por gNMIC y las expone para consulta via PromQL.

## Acceso

- **URL**: http://localhost:9090
- **IP de Gestión**: 10.99.1.10

## Configuración

El archivo `configs/prometheus/prometheus.yml` define el scraping de gNMIC.
