# Grafana

## Descripción

Grafana visualiza las métricas almacenadas en Prometheus mediante dashboards pre-configurados.

## Acceso

- **URL**: http://localhost:3030
- **Credenciales**: admin/admin
- **IP de Gestión**: 10.99.1.11

## Dashboards Pre-configurados

| Dashboard | Descripción |
|-----------|-------------|
| SROS Dashboard | Métricas de BNG MASTER y SLAVE |
| Small ISP SR Linux Edge | Métricas de OLT, Carrier 1 y Carrier 2 |
| Nokia Syslog Overview | Logs de SR OS y SR Linux vía Alloy + Loki |
